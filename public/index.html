LLM Extraction & Auditing

Both stages of AI extraction use state-of-the-art Large Language Models with precise engineering configurations for high accuracy.

Model: claude-haiku-4-5-20251001
Temperature: 0 (Deterministic)
Prompting style: XML tag encapsulation
Validation: Pydantic runtime checks

AI Flow: Plaintext is extracted from the PDF, segmented, and wrapped in structured XML tags before sending to Claude. The structured JSON returned by Claude is parsed by a Python Pydantic BaseModel to validate type safety at runtime.

Infisical Vault Integration

We use Infisical, an open-source secrets manager, to securely store and inject API credentials. No secrets are hardcoded or committed to git.

1 Secrets fetched from Infisical cloud dashboard vault.
2 Injected into worker context at runtime with infisical run --.
3 Claude API receives token directly in memory, eliminating filesystem leaks.

This ensures enterprise-grade security for keys like ANTHROPIC_API_KEY and TRIGGER_SECRET_KEY across local runs and cloud deployments.
